# 📝 Markdown Cheat Sheet

## 📚 Tiêu đề (Headings)

```markdown
# Tiêu đề cấp 1
## Tiêu đề cấp 2
### Tiêu đề cấp 3
#### Tiêu đề cấp 4
```

---

## ✍️ Định dạng văn bản

```markdown
**In đậm**
*In nghiêng*
~~Gạch ngang~~
`Đoạn mã ngắn`
```

---

## 📋 Danh sách

### ✅ Danh sách không thứ tự

```markdown
- Mục 1
- Mục 2
  - Mục con
* Dấu * cũng được dùng
```

### 🔢 Danh sách có thứ tự

```markdown
1. Bước đầu tiên
2. Bước tiếp theo
3. Bước cuối cùng
```

---

## 🔗 Liên kết và hình ảnh

```markdown
[Text hiển thị](https://example.com)

![Alt text](https://example.com/image.jpg)
```

---

## 📦 Code block

### ✅ Inline code

```markdown
Sử dụng `npm install` để cài đặt.
```

### ✅ Block code nhiều dòng

<pre>
```js
function sayHello() {
  console.log("Hello Markdown!");

```
</pre>
